What it does

Core Features

📐
Plane Detection
Real-time horizontal surface detection using ARCore raycasting to anchor 3D objects accurately in physical space.
✋
Gesture Controls
Touch-based interactions for placing, scaling (pinch), rotating (twist) and moving objects on the detected plane.
⚡
Performance Optimised
Lightweight 3D models and efficient update loops to maintain smooth AR frame rates on mid-range Android devices.
🎯
Object Tracking
Persistent object placement that stays locked to the real-world surface even as the camera moves around.
How it works

Technical Flow

01
ARSession Initialisation
AR Foundation initialises ARCore on device launch, starts camera feed and begins scanning the environment for trackable planes.
02
Raycast Hit Detection
A screen-centre raycast is fired every frame against detected AR planes. When a stable hit is found, a placement indicator appears.
03
Object Instantiation
On tap, the selected 3D prefab is instantiated at the raycast hit pose, aligned to the plane normal for natural placement.
04
Gesture Manager
A custom C# GestureManager component tracks multi-touch input to translate pinch distance into scale and rotation delta into Y-axis spin.
PlacementController.cs
void Update() {
  if (arRaycastManager.Raycast(screenCenter, hits,
      TrackableType.PlaneWithinPolygon)) {
    var hitPose = hits[0].pose;
    placementIndicator.SetActive(true);
    placementIndicator.transform.position = hitPose.position;
    placementIndicator.transform.rotation = hitPose.rotation;
  }
}

void PlaceObject() {
  Instantiate(selectedPrefab,
    placementIndicator.transform.position,
    placementIndicator.transform.rotation);
}
